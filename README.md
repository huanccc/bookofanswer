# 🔮 数字神谕 | Digital Oracle (答案之书)

> 在心中默念你的问题。系统将通过算力与概率，为你解析最终的答案。

这是一个基于纯前端技术栈（Vanilla HTML/CSS/JS）构建的**极简先锋风 / 赛博朋克风**互动网页。它不仅仅是一个随机抽取答案的工具，更是一场充满数字艺术气息的视觉与听觉体验。

## ✨ 核心特性

* **🖥️ ASCII 动态交互场**：背景采用 HTML5 Canvas 实时渲染 ASCII 字符构成的“数字山脉”与“噪点”，并带有趣味的鼠标排斥与放大镜物理反馈。
* **🔣 赛博乱码解码特效 (Text Scramble)**：每次抽取答案时，文字会像黑客帝国或科幻电影中一样，经历酷炫的乱码翻滚后最终定格。
* **🎛️ 原生 Web Audio 电子音效**：纯用 JavaScript 合成的克制且高级的“机械电子滴答声”，无需加载任何外部音频文件。
* **🖱️ 极简自定义物理光标**：去除了系统默认光标，替换为带有阻尼延迟特效的极简圆点与轮廓线。
* **📺 复古监视器美学**：界面融合了工程网格背景与隐约的 CRT 屏幕扫描线（Scanlines）特效，视觉饱满且克制。

## 🛠️ 技术栈

* **HTML5**: 语义化结构与 Canvas 画布
* **CSS3**: 极简设计、Flex/Grid 布局、CSS 渐变与扫描线特效
* **JavaScript (ES6+)**: 动画渲染引擎、Web Audio API 发声、DOM 交互与时间戳计算
* **零依赖**: 无需 React、Vue 或任何外部组件库，大道至简。

## 🚀 如何运行与部署

### 本地运行
本作天生具备极高的便携性，**无需配置任何本地服务器**：
1. 下载或克隆本项目。
2. 双击打开 `index.html` 文件即可在浏览器中体验。

### 公网部署
推荐使用 [Netlify Drop](https://app.netlify.com/drop) 或 [Zeabur](https://zeabur.com/)：
* 直接将整个文件夹拖拽至 Netlify，3秒钟即可获取公网分享链接。
* 或将其推送到 GitHub 后，在 Zeabur 一键导入部署，享受国内直连的畅快体验。
